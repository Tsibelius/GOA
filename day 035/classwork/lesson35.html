Python და HTML კითხვები

1. რა არის Python?
Python არის პოპულარული პროგრამირების ენა, რომელიც ცნობილია მისი სიმარტივით და კოდის გასაგები სტილით.
2. რა არის HTML?
HTML (HyperText Markup Language) არის ენა, რომელიც გამოიყენება ვებგვერდების სტრუქტურისა და შინაარსის შესაქმნელად.
3. რა ფუნქცია აქვს Python-ში "def"-ს?
"def" გამოიყენება Python-ში ფუნქციის განსაზღვრისთვის.
4. რა არის HTML tag-ი?
HTML tag არის სპეციალური კოდი, რომელიც განსაზღვრავს ელემენტებს ვებგვერდზე.
5. როგორ იქმნება ცვლადი Python-ში?
Python-ში ცვლადი იქმნება მარტივად, მაგალითად: x = 5.
6. როგორ იწყება HTML დოკუმენტი?
HTML დოკუმენტი იწყება <!DOCTYPE html> და <html> tag-ებით.
7. რა ფუნქცია აქვს if-else კონსტრუქციას Python-ში?
if-else კონსტრუქცია გამოიყენება პირობების შესამოწმებლად და შესაბამისი ქმედებების შესასრულებლად.
8. რა არის HTML-ის მთავარი ელემენტები?
HTML-ის მთავარი ელემენტებია: <html>, <head>, <body>.
9. რა ფუნქცია აქვს for loop-ს Python-ში?
for loop გამოიყენება განმეორებითი ოპერაციების შესასრულებლად სიებზე, სტრიქონებზე და სხვა iterable ობიექტებზე.
10. როგორ ხდება სურათის დამატება HTML-ში?
სურათის დამატება ხდება <img> tag-ის საშუალებით, სადაც src ატრიბუტი განსაზღვრავს სურათის მისამართს.
11. როგორ გამოიყენება list-ები Python-ში?
List არის მონაცემთა სტრუქტურა, რომელიც შეიცავს ელემენტების ან ჩამონათვალის შეკრებას. ის დინამიკურად იზრდება და მასში შეიძლება ინახებოდეს სხვადასხვა ტიპის მონაცემები.
12. რა არის HTML-ის input ელემენტი?
HTML-ის input tag გამოიყენება მომხმარებლისგან ინფორმაციის მისაღებად ფორმის ელემენტში.
13. რა განსხვავებაა tuple-სა და list-ს შორის Python-ში?
List არის mutable, ანუ შეიძლება შეიცვალოს, ხოლო tuple არის immutable და არ ექვემდებარება ცვლილებებს.
14. როგორ ემატება ლინკი HTML-ში?
ლინკის დასამატებლად გამოიყენება <a> tag. მაგალითად: <a href="http://example.com">Click Here</a>.
15. როგორ იქმნება ფუნქცია Python-ში?
Python-ში ფუნქცია იქმნება def კვანძის გამოყენებით. მაგალითად: def my_function():.
16. რა ფუნქცია აქვს CSS-ს HTML-ში?
CSS (Cascading Style Sheets) გამოიყენება ვებგვერდის ვიზუალური მხარის სტილიზაციისთვის, მათ შორის ტექსტის, ფერის, ფონტის და განლაგების განსაზღვრაში.
17. რა არის dictionary Python-ში?
Dictionary არის მონაცემთა სტრუქტურა, რომელიც ინახავს key-value წყვილებს. მაგალითად: my_dict = {'key': 'value'}.
18. რა მნიშვნელობა აქვს div tag-ს HTML-ში?
div tag გამოიყენება ვებგვერდზე კონტენტის ბლოკებად დასაყოფად, რაც მას არქიტექტურულად მართვადს ხდის.
19. როგორ ხდება გამონაკლისების (exceptions) დამუშავება Python-ში?
გამონაკლისების დასამუშავებლად გამოიყენება try-except ბლოკები, რომლებიც ცდილობენ კოდის ნაწილის შესრულებას და გამონაკლისის შემთხვევაში შესაბამის ქმედებას ასრულებენ.
20. როგორ ემატება ვიდეო HTML გვერდს?
ვიდეო HTML გვერდს ემატება <video> tag-ის გამოყენებით. მაგალითად: <video src="video.mp4" controls></video>.
21. რა არის Lambda ფუნქცია Python-ში?
Lambda ფუნქცია Python-ში არის მოკლე, ერთჯერადი ფუნქცია, რომელიც შესაძლებელია ერთ ხაზზე განისაზღვროს. მაგალითად: lambda x: x + 1.
22. რა ფუნქცია აქვს tag-ს HTML-ში?
tag გამოიყენება გვერდის მეტაინფორმაციის დასამატებლად, მაგალითად, გვერდის აღწერილობისა და კოდირების ტიპის განსაზღვრაში.
23. როგორ გამოიყენება while loop Python-ში?
while loop განმეორებით ასრულებს კოდს მანამ, სანამ წინასწარ განსაზღვრული პირობა ჭეშმარიტია.
24. რა განსხვავებაა id-სა და class-ს შორის HTML-ში?
id უნიკალურია თითოეული ელემენტისთვის და გვერდზე მხოლოდ ერთხელ გამოიყენება, class შეიძლება ერთზე მეტი ელემენტისთვის იყოს გამოყენებული.
25. როგორ გამოიყენება Python-ში list comprehension?
List comprehension არის სიმარტივის მეთოდი სიების შესაქმნელად ლაკონური ფორმით. მაგალითად: [x for x in range(10)].
26. როგორ ხდება სიის შექმნა HTML-ში?
სიის შესაქმნელად გამოიყენება <ul> (unordered list) ან <ol> (ordered list) tag-ები.
27. როგორ ინახება მონაცემები ფაილში Python-ში?
Python-ში მონაცემების ფაილში შესანახად გამოიყენება open() ფუნქცია, რომელიც ხსნის ფაილს და write() ფუნქცია მონაცემების ჩასაწერად.
28. რა არის HTML-ის form ელემენტი?
Form tag გამოიყენება მომხმარებლისგან მონაცემების მისაღებად. მასში შეიძლება იყოს input, checkbox, radio, და სხვა ელემენტები.
29. როგორ მუშაობს moduli Python-ში?
Moduli არის დამატებითი პაკეტები და ბიბლიოთეკები, რომლებიც Python-ში დამატებით ფუნქციონალს უზრუნველყოფენ. ისინი გამოიყენება import კვანძით.
30. რა ფუნქცია აქვს tag-ს HTML-ში?
Table tag გამოიყენება მონაცემების ცხრილის სახით გამოსახვისთვის, სადაც გამოიყენება <tr>, <th> და <td> ელემენტები.
31. როგორ ხდება ფუნქციის გადაცემული არგუმენტების დამუშავება Python-ში?
ფუნქციები იღებენ არგუმენტებს და მათ დამუშავებას შესაბამისი სახელებით, რომლებიც განსაზღვრულია ფუნქციის დეფინიციაში.
32. როგორ ემატება სტილი HTML ელემენტს?
HTML ელემენტებს სტილი ემატება CSS-ის მეშვეობით. მაგალითად, <p style="color: red;">.
33. რა არის class-ები და ობიექტები Python-ში?
Class-ები არის ობიექტების ბლუპრინტი, ხოლო ობიექტები კლასის ეგზემპლარები. ისინი გამოიყენება ობიექტზე ორიენტირებულ პროგრამირებაში.
34. რა არის HTML-ის
tag?
Section tag გამოიყენება დოკუმენტის ლოგიკური სექციების შექმნისთვის, რაც კონტენტს თემატურად ყოფს.
35. როგორ გამოიყენება map ფუნქცია Python-ში?
map ფუნქცია საშუალებას გაძლევთ გამოიყენოთ ფუნქცია ყველა ელემენტზე iterable-ში. მაგალითად, map(lambda x: x*2, [1, 2, 3]).

37. როგორ ხდება მონაცემების ტიპების გადამოწმება Python-ში?
Python-ში მონაცემის ტიპის გადამოწმება ხდება type() ფუნქციით. მაგალითად: type(5) დაბრუნდება int.
38. როგორ ემატება ფაილი ატვირთვის ფუნქცია HTML ფორმას?
ფაილის ატვირთვისთვის HTML-ში გამოიყენება <input type="file"> ელემენტი.
39. რა არის list slicing Python-ში?
List slicing არის სიიდან ელემენტების ამოღება გარკვეული ინდექსებით. მაგალითად: my_list[1:4] ამოიღებს ელემენტებს მეორე ინდექსიდან მეოთხემდე.
40. რა ფუნქცია აქვს HTML-ის
tag-ს?
Nav tag გამოიყენება ნავიგაციის ბლოკის განსაზღვრად, რომელიც შეიცავს ბმულებს გვერდის სხვა ნაწილებთან ან გარე რესურსებთან.
41. როგორ გამოიყენება set მონაცემთა სტრუქტურა Python-ში?
Set არის უნიკალური მნიშვნელობების კოლექცია, რომელიც არ შეიცავს დუბლირებულ ელემენტებს. მაგალითად: my_set = {1, 2, 3}.

43. როგორ მუშაობს Python-ში გლობალური ცვლადები?
გლობალური ცვლადები ინახება პროგრამის გარე სივრცეში და მათზე წვდომა შესაძლებელია ნებისმიერი ფუნქციიდან, თუმცა მათი ცვლილება ხდება global კვანძით.
44. როგორ ხდება ბმული გარე ფაილებთან HTML-ში?
გარე ფაილების ჩასართავად გამოიყენება <link> tag ან <script> tag, რომელიც სქრიპტების დასამატებლად გამოიყენება.
45. როგორ მუშაობს Python-ში რეგულარული გამოხატულებები (regex)?
რეგულარული გამოხატულებები გამოიყენება ტექსტის პეტერნების დასაჭერად ან შესამოწმებლად, re მოდულის მეშვეობით.
